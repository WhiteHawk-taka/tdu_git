gitのセットアップ

Windowsの方

git for Windowsのサイトからダウンロードしてインストールしておいてください。

インストール手順

  • Select Components はデフォルトのままで結構です。(git GUIはチェックを外しても大丈夫です。)
  • Adjusting your PATH environment はRun Git from the Windows Command Promptを選択してください。
  • Configuring the line ending conversions はデフォルトのままで結構です。
  • Configuring the terminal emulate to use with Git Bash は2つめのUse Windows' default console windowを選択してください。

このブログを参考にすると良いかもしれません。
